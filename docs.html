Quick Start

Get started with NCS-API in minutes. Here's a simple example to cluster your first dataset:

// Initialize NCS-API client
const ncsApi = new NCSClient({
  apiKey: 'your_api_key_here',
  baseURL: 'https://api.ncs-cluster.com/v1'
});

// Prepare your data
const data = [
  { x: 1.0, y: 2.0 },
  { x: 1.5, y: 1.8 },
  { x: 5.0, y: 8.0 },
  { x: 8.0, y: 8.0 },
  { x: 1.0, y: 0.6 },
  { x: 9.0, y: 11.0 }
];

// Run clustering
const result = await ncsApi.cluster({
  data: data,
  algorithm: 'kmeans',
  clusters: 3,
  options: {
    maxIterations: 100,
    tolerance: 0.001
  }
});

console.log('Clusters:', result.clusters);
console.log('Centroids:', result.centroids);
console.log('Quality Score:', result.silhouetteScore);

Clustering API

The core clustering endpoint supports multiple algorithms and customization options.

POST /v1/cluster

Perform clustering analysis on your dataset using various algorithms.

Request Body

data array Array of data points with x, y coordinates
algorithm string Clustering algorithm: "kmeans", "dbscan", "hierarchical"
clusters integer Number of clusters (for k-means)
options object Algorithm-specific options

Response

{
  "status": "success",
  "executionTime": 25.3,
  "algorithm": "kmeans",
  "clusters": [...],
  "metrics": {
    "silhouetteScore": 0.847,
    "inertia": 15.23,
    "iterations": 12
  }
}

Error Handling

NCS-API uses standard HTTP status codes and provides detailed error messages:

400
Bad Request

Invalid request parameters or malformed data

401
Unauthorized

Invalid or missing API key

429
Rate Limited

Too many requests, slow down

500
Server Error

Internal server error, try again later

Error Response Format

{
  "status": "error",
  "code": 400,
  "message": "Invalid data format",
  "details": {
    "field": "data",
    "issue": "Expected array of objects with x, y coordinates"
  },
  "requestId": "req_abc123def456"
}
